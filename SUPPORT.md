# Support

AIAGET is an open source project. Community support is handled through GitHub.

- Use issues for reproducible bugs and small enhancement requests.
- Use discussions, when enabled, for usage questions and design proposals.
- Do not share credentials, production data, customer files, or private logs in
  public issues.
- Report security vulnerabilities through the private process in
  [SECURITY.md](./SECURITY.md).

For operational deployments, include the service versions, deployment mode,
relevant environment variable names, and sanitized logs when opening an issue.
