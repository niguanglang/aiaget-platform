# M131 续约机会调账反向追踪

## 目标

M130 已经让账单调账记录可以回链到续约机会。M131 补齐反向链路：在续约机会详情页直接展示由该机会生成的成交入账调账摘要，方便客户成功、财务运营和审计人员从机会详情进入账单与审计追踪。

## 范围

- `CustomerSuccessOpportunityDetail` 增加只读字段 `billing_adjustments`。
- `GET /customer-success-opportunities/:id` 查询当前机会来源为 `CUSTOMER_SUCCESS_OPPORTUNITY` 的未删除调账记录。
- 续约机会详情页“成交入账闭环”卡片展示关联调账单号、状态、金额、账单号、入账时间和原因。
- 详情页提供两个入口：
  - `调账记录`：进入 `/billing/adjustments` 继续财务处理。
  - `审计追踪`：进入 `/audit` 按调账单号检索审计事件。

## 页面边界

- 续约机会详情页只展示与当前机会强关联的调账摘要。
- 调账审批、应用、作废仍归属账单调账页。
- 列表页和分析页不引入成交入账、调账审批或调账详情能力。

## 验收点

- 详情 API 返回当前机会关联调账数组。
- 已赢单机会能在详情页看到关联调账摘要。
- 无关联调账时仍保留原有成交入账表单和提示。
- 列表页、分析页不出现调账反向追踪详情。
