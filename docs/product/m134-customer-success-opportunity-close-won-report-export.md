# M134 续约机会成交复盘报告导出

## 背景

M133 已经提供成交复盘报告详情页。M134 补齐只读导出能力，让客户成功、财务运营和审计角色可以把报告下载为 Markdown 文件，用于离线复盘、会议纪要、财务核对和审计留痕。

## 范围

- 后端新增 `GET /customer-success-opportunities/:id/close-won-report/export`。
- 导出格式为 `text/markdown; charset=utf-8`。
- 前端在 `/customer-success-opportunities/[id]/close-won-report` 页新增“导出报告”按钮。
- 导出内容复用现有成交复盘报告数据，不新增表结构、不写对象存储。

## 页面职责

- 导出入口只放在成交复盘报告页。
- 续约机会列表页不导入导出 API，不持有报告导出状态。
- 续约机会分析页不导入导出 API，不承担单个报告职责。
- 导出是只读能力，不需要确认弹窗，也不触发成交入账、跟进行动或编辑动作。

## Markdown 内容

- 报告标题与客户摘要。
- 客户价值复盘。
- 来源链路。
- 入账追踪。
- 复盘要点。
- 下一步动作。

## 验收

- 后端服务测试覆盖 Markdown 导出内容。
- 前端 IA 契约确认导出能力只属于成交复盘报告页。
- 导出按钮有导出中禁用状态和失败提示。
- 文件名以 `.md` 结尾。
- Control API 与 Web typecheck 通过。
