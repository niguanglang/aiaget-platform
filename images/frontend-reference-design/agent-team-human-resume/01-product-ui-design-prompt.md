# Product UI Design Image Prompt

为一个企业 AI Agent 平台设计 `/agent-teams` 页面里的“人工介入审批与恢复执行”高保真产品界面。界面语言为中文，风格是极简、科技、高级、产品感强的企业 SaaS 控制台。

页面已存在 Agent 协作中心、运行记录选择器、步骤时间线、运行详情、接力记录、反馈面板。本次设计只在现有运行轨迹工作区内增强人工介入区域，不做营销页。

布局要求：
- 使用 Dashboard Layout 与 Bento Grid 组织信息。
- 主区域左侧为运行摘要和步骤时间线，右侧为人工动作面板。
- 在运行状态为“等待人工介入”时，突出一个“人工介入”卡片，展示待处理接力原因、来源成员、目标成员、创建时间、审批备注输入框。
- 卡片底部有两个操作按钮：通过并继续、拒绝并结束。
- 接力记录列表中 PENDING 项展示待审批状态，APPROVED/REJECTED/AUTO 展示历史状态。

视觉要求：
- Tailwind/shadcn 风格，细边框、轻阴影、backdrop blur、soft shadow。
- 使用浅色企业控制台背景，少量蓝色/青色作为状态强调，避免廉价发光和过度渐变。
- 状态徽标清晰区分：等待人工介入、已通过、已拒绝、自动接力。
- 信息密度适中，留白充足，文本不溢出。

交互状态：
- 无权限时按钮禁用并显示“无审批权限”。
- 审批中显示按钮 loading/pending 状态。
- 没有待处理接力时显示空状态。
- 保留监控追踪、复制 Trace、提交接力、保存反馈等已有入口。

不要使用 Emoji，不要大圆堆叠，不要信息过满，不要夸张 3D。允许非常克制的玻璃质感和微弱渐变网格背景。
