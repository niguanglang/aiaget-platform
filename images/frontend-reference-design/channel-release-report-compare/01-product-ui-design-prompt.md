# Product UI Design Image Prompt

为一个企业级 AI Agent 平台设计 `/channels/release/reports` 渠道发布复盘报告页面，高保真 SaaS 后台界面，中文文案。

页面目标：运营人员选择发布渠道后查看当前发布复盘摘要、风险建议、复盘快照列表、单个快照详情，以及两个快照之间的版本对比。

视觉风格：极简、科技、企业产品感；白底、细边框、轻阴影、少量玻璃质感；布局采用 Dashboard/Bento Grid，留白充足，避免过度渐变和夸张发光。按钮与状态徽标使用 shadcn/ui 风格，卡片圆角克制。

信息架构：
- 顶部：发布复盘报告标题、路由副标题、刷新状态和刷新按钮。
- 渠道选择区：横向展示渠道名称、渠道类型、发布状态、健康状态。
- 报告摘要卡片：展示渠道、结论、事件等级、健康状态、发布状态、报告窗口，并提供“创建快照”按钮。
- 报告正文：深色代码块样式展示 markdown 复盘正文。
- 风险建议卡片：按严重程度展示风险标题和建议。
- 复盘快照列表：每条只展示结论、创建时间、快照 ID、事件等级和三个行内操作“查看快照 / 设为基准 / 设为对比”。
- 下方双栏：左侧为快照详情，包含快照元数据、归档报告正文、来源事件；右侧为报告版本对比，包含基准快照、对比快照、变化统计和按“摘要差异 / 指标差异 / 风险差异 / 时间线差异”分组的差异项。

交互状态：加载、暂无数据、错误提示、禁用按钮、创建成功提示、无权限禁用、两个快照相同时不能对比。所有按钮文案和提示必须为中文。
