# Product UI Design Image Prompt

High-fidelity Chinese enterprise SaaS admin detail page for an AI Agent Platform Conversation Center, route `/conversations/[id]`. Show a focused conversation detail workspace: compact header with back navigation, conversation status badge, latest run status badge, title, agent name, user email, and right-aligned destructive archive action. Below header show four metric cards: 消息、运行、反馈、最近状态. Main layout uses two columns: left large 消息流 card with scrollable message list, Chinese role labels, timestamps, textarea reply box, streaming progress state, and 继续会话 button; right 运行轨迹 card with run status, model, latency, token, cost and step timeline. Lower row: 反馈 card with rating selector, comment textarea, history list; 引用与工具调用 card with references and tool call summaries including approval request links. Use shadcn/Tailwind style, subtle borders, soft shadows, clean Chinese labels, no emoji, no heavy gradients.
