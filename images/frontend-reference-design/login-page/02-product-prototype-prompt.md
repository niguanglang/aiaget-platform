# Product Prototype / Wireframe Prompt

Create a low- to mid-fidelity prototype for the same `/login` page.

Prototype requirements:
- Show a two-column responsive layout
- Top-left brand row
- Left column with headline, supporting paragraph, trust pills, and a compact bento capability grid
- Right column with the login form card
- Login card sections: badge, title, description, tenant field, email field, password field with eye toggle, error area, primary button, development credential helper
- Preserve clear spacing and hierarchy
- Mark background atmosphere as a light grid + orb + particles only, with no extra modules or unsupported actions
