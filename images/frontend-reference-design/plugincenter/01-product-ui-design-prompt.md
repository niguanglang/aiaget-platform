# Product UI Design Image Prompt

生成一张企业级 SaaS 后台「插件生态中心」高保真产品 UI 设计图，中文界面，16:10 桌面端视口。页面路由是 `/plugins`，用于管理插件市场、租户安装实例、插件启停升级、菜单注入、Hook 绑定、权限预览和审计记录。

界面结构：
- 顶部为控制台内容区标题，不做营销 Hero。标题为「插件生态中心」，副文案说明「管理插件市场、租户安装实例、启停升级、菜单注入、Hook 和审计记录」。
- 标题下方是 4 个指标卡：插件总数、已启用、待审核、菜单注入。
- 主体使用 Dashboard / Bento Grid：左侧大卡是插件列表，右侧详情卡是当前插件安装实例。
- 插件列表顶部有分段控制「市场 / 已安装」，搜索框、状态筛选、风险筛选、清空按钮。
- 表格列包含：插件、版本、状态、风险、能力、操作。市场行提供「安装」或「详情」，已安装行提供详情、启用/停用、升级。
- 详情卡顶部显示状态徽标、插件名称、编码、提供方、来源；右侧有「配置」「启用/停用」「升级」按钮。
- 详情内容包括：版本信息、安全预览、菜单注入卡片、Hook 绑定卡片、权限预览标签、审计记录列表。

视觉风格：
- 极简、科技、高级、产品感强；白色/浅灰为主体，辅以克制的蓝色、青绿色和琥珀状态色。
- 使用细边框、轻阴影、backdrop blur、低透明度网格与细粒噪声纹理。
- 右后方可出现非常低对比度的 3D particle field / wireframe cube / lattice 背景，但只作为氛围，不能遮挡内容。
- 信息密度适中，留白充足，表格和详情卡清晰可扫读。
- 所有文字必须是中文；不要 Emoji；不要廉价强发光；不要大圆堆叠；不要厚重紫色渐变；不要让 3D 元素抢主体。

组件细节：
- Button 使用小尺寸图标+中文文本。
- 状态使用小型圆角徽标，风险颜色克制。
- 卡片圆角不超过 8px。
- 列表 hover 有轻微反馈。
- 整体像真实可运营的企业 Agent 平台插件中心，而不是模板页面。
