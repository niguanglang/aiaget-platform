# Product UI Design Image Prompt

为一个企业级 AI Agent 平台设计“生产落地中心”后台页面，高保真产品 UI，中文界面。

页面目标：集中展示上线前验收状态，覆盖环境配置、外部服务、第三方联调、发布验收、风险项。整体是成熟 SaaS 控制台，不是营销页。

布局要求：
- 左侧仍是系统原有后台导航，主体为 Dashboard Layout。
- 顶部为页面标题“生产落地中心”、只读状态、生成时间、刷新按钮和监控中心入口。
- 顶部指标卡横向展示：检查项、已就绪、人工验收、阻塞项、落地分。
- 主体采用左侧分组锚点导航 + 右侧分组卡片。
- 每个分组卡内是紧凑检查项列表，每行包含状态徽标、风险级别、负责人、说明、证据行、跳转按钮。

视觉风格：
- 极简、高级、真实产品感；白色/浅灰背景，细边框，轻阴影，backdrop blur。
- 使用克制的青绿色和蓝色作为状态/信息强调，红色只用于阻塞风险，避免廉价发光。
- 可有 subtle gradient mesh 背景，但不能抢主体。
- 卡片圆角不超过 8px，信息密度适中，留白充足。

交互状态：
- READY 显示“已就绪”，WARNING 显示“需关注”，BLOCKED 显示“阻塞”，MANUAL 显示“人工验收”。
- 页面只读，不出现保存、创建、删除类按钮。
- 空/加载/错误状态需要有专业提示。

禁止：
- 不要 Hero Landing，不要大圆堆叠，不要 Emoji，不要过度渐变，不要把真实凭证明文展示出来。
