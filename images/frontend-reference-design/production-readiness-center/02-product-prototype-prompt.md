# Product Prototype / Wireframe Prompt

设计一个中保真线框原型，用于企业 AI Agent 平台的“生产落地中心”。

信息架构：
1. 页面头部：标题、说明、状态标签、生成时间、刷新、监控中心。
2. 指标区：5 个小指标卡，分别为检查项、已就绪、人工验收、阻塞项、落地分。
3. 左侧锚点：环境配置、外部服务、第三方联调、发布验收、风险项，每个显示分类检查项数量和状态计数。
4. 右侧内容：按分类展示检查项卡片。
5. 检查项卡片字段：标题、状态、风险级别、负责人、说明、证据、操作入口。

页面边界：
- 这是只读验收看板，不是配置表单页。
- 配置操作跳转到已有模块，例如存储设置、模型中心、API Key、渠道发布、插件中心、知识库健康、监控中心、安全中心、成本中心。
- 阻塞项需要突出，但不要弹窗或强制流程。

原型风格：
- 后台产品线框，密集但清晰。
- 不使用营销 hero。
- 列表不超过核心字段，证据短句分行展示。
