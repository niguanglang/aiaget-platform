# Product UI Design Image Prompt

High-fidelity Chinese enterprise SaaS admin detail page for an AI Agent Platform Prompt Center, route `/prompts/[id]`. Show a compact prompt detail header with back navigation, status/version/type badges, prompt name, description, and right actions: 编辑 (route navigation), 复制, 发布, 删除. Use clean shadcn/Tailwind dashboard style with subtle borders and soft shadows. Below header show metrics cards for 变量、版本、测试、智能体. Main two-column layout: left side has 提示词编辑器 content card with save content button, 变量 table card, 版本 table card with change note and rollback actions. Right side has 模板资料 card, 渲染与测试 card with JSON inputs, render/test buttons, rendered output and real model test result, 最近测试 card, 智能体引用 card, 活动 card. Chinese labels only; avoid embedding full basic edit form in detail page.
