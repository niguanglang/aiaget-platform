# Product Prototype / Wireframe Prompt

Mid-fidelity wireframe for Prompt Center detail route `/prompts/[id]`, Chinese admin console. Focus on page responsibilities: detail header, route edit action, metrics, content editor, variables, versions, render/test, recent tests, agent references, activity, delete confirmation. Explicitly show that full metadata edit form is not present on the detail page and is a separate `/prompts/[id]/edit` page. Include empty states for variables, versions, tests, and disabled write controls when lacking permission.
