# Product UI Design Image Prompt

High-fidelity Chinese enterprise SaaS admin detail page for an AI Agent Platform Tool Center, route `/tools/[id]`. Show a clean tool detail workspace with a compact header containing back navigation, status badges, risk badge, method badge, title, code, description, and right-aligned actions: 编辑, 复制, 启用/停用, 删除. Use Tailwind/shadcn style: white cards, subtle borders, soft shadows, restrained blue/teal accents, glass-like but business-focused. Below header show four metric cards, then a two-column dashboard layout: HTTP 配置 card, 鉴权与风险策略 card, 输入结构 JSON card, 输出结构 JSON card, 测试面板 with JSON textarea and run result preview, 调用日志 timeline/list, 智能体引用 card, 使用摘要 card. Chinese labels only. Avoid marketing hero, emoji, excessive gradients, large glowing effects, and overloaded tables.
