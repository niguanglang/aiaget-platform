# Product UI Design Image Prompt

高保真企业后台界面设计，一组现代 SaaS 控制台页面局部截图拼板，主题为“运营字段口径清晰、无营销文案、无开发说明、无占位按钮”。画面包含左侧若依式多级菜单、顶部标题区、紧凑状态标签、指标卡片、筛选栏、表格和行内操作。页面标题使用中文业务名称，例如“模型中心、工具中心、知识库中心、会话中心、监控中心、成本总览”。副标题只描述真实字段：状态、数量、最近调用、风险等级、Trace、成本、权限范围。无 Mxx 里程碑编号，无“列表页用于/独立页面/已拆分”等内部说明。无权限时隐藏新增按钮，只保留只读提示。视觉风格极简、科技、高级、产品感强，白色背景、细边框、轻阴影、克制蓝绿点缀，按钮数量少，低频操作收进行内区域。
