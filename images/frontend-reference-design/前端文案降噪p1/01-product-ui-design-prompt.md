# Product UI Design Image Prompt

高保真企业后台产品界面，主题为“企业 Agent 平台运营控制台文案降噪”。画面展示一个极简科技感 SaaS 管理后台，左侧为深浅适中的可折叠菜单，主区域为渠道中心、安全中心、平台事件与用量、系统设置的多页面缩略组合。每个页面只显示业务字段、状态、关键指标和明确操作入口，不出现里程碑编号、内部改造说明、营销口号、占位按钮。视觉使用 Tailwind 风格的白色卡片、细边框、轻阴影、少量 backdrop blur、冷静蓝灰主色和少量状态色。按钮分布清晰：顶部放全局操作，行内放单条记录操作，无权限动作隐藏或转为只读说明。所有可见文字为中文，界面像真实可运营产品，不像模板或方案展示页。
