# Product Prototype / Wireframe Prompt

低保真产品原型线框图，展示企业 Agent 平台的页面降噪规则：左侧菜单进入渠道中心、安全中心、平台事件与用量、系统设置；每个列表页包含标题、简短业务说明、筛选区、指标卡、列表卡片、分页和行内操作；详情页包含基础信息、关联数据、日志和 JSON 预览；配置页包含参数分组和保存/恢复默认动作。重点标注信息层级：列表页只呈现核心识别字段和状态，详情页呈现完整上下文，表单页承载复杂配置。无权限时页面显示权限提示，不显示禁用的新建、编辑、查看链接。所有文案为中文业务语言，不出现“拆分、独立页、完整表单、Mxx、闭环、赋能、打造”等内部或营销化表达。
