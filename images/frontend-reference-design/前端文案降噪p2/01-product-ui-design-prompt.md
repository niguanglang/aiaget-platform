# Product UI Design Image Prompt

高保真企业后台产品界面，主题为“企业 Agent 平台低频业务模块文案降噪”。画面展示客户评估、方案包、交付复盘、成果资产、客户成功计划、行动、续约机会、Skill、插件详情等页面。整体为极简科技感 SaaS 控制台，白色卡片、细边框、轻阴影、少量玻璃质感、克制状态色。列表页只呈现业务对象名称、状态、评分、负责人、来源关系和行内操作；详情页呈现业务上下文和关联记录；表单页呈现结构化录入。所有可见文案为中文业务语言，不出现“列表页用于、只保留、完整内容、进入详情页、沉淀、赋能、闭环、Mxx”等内部或营销化表达。无权限用户不看到灰色占位新建或编辑按钮，只看到必要的权限提示。
