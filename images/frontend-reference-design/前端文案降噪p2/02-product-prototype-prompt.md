# Product Prototype / Wireframe Prompt

低保真产品原型线框图，展示企业 Agent 平台低频业务模块的页面职责：客户评估列表、方案包列表、交付复盘列表、成果资产列表、客户成功计划列表、行动列表、续约机会列表、Skill 列表和插件详情。每个列表页包含标题、短业务说明、指标卡、筛选区、表格或列表卡、分页、行内查看/编辑/归档；详情页包含基础信息、业务信息、来源关系、操作记录和关联入口；新增/编辑页使用独立表单面板。无权限时隐藏新建和编辑入口，保留可读数据和权限提示。重点标注文案规则：只用对象、字段、状态、动作描述，不展示 IA 改造说明、方案口号、里程碑编号或占位按钮。
