# Product UI Design Image Prompt

高保真企业后台界面，主题为“企业 Agent 平台控制台文案降噪后的真实产品页面”。画面展示一个现代 SaaS 管理后台：左侧为多级菜单，右侧主内容包含开放接口文档、资源授权校验、业务创建表单、审计列表和存储文件列表的局部拼接。所有标题和说明都使用简洁中文业务语言，不出现 Mxx、里程碑、模板说明、占位按钮、营销化口号。视觉风格克制、极简、科技感强；Tailwind 风格，细边框、轻阴影、轻微 backdrop blur，白色和浅灰为主，蓝色作为主色，状态标签清晰但不过度发光。列表只显示核心字段，详情和配置入口以普通业务按钮呈现。不要使用 Emoji，不要使用夸张渐变，不要展示“开发中”“示例插件”“模拟检查”等字样。
