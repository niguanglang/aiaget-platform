# Product Prototype / Wireframe Prompt

Create a wireframe for a Chinese enterprise admin list page. Layout: page title and primary action in one row, metric strip, filter bar with search/select controls, table with key fields only, pagination, confirmation modal for destructive action. Do not include introductory copy, concept tags, slogans, feature descriptions or usage instructions. The prototype should feel like an internal operations console similar to a mature ERP/admin system.
