# Product UI Design Image Prompt

Create a high-fidelity enterprise SaaS admin detail page card for a Chinese “企业 AI Agent 平台” customer success opportunity detail route. The page already has a clean dashboard layout with white glass cards, thin borders, subtle shadows, soft backdrop blur, restrained blue/emerald accents, and Chinese UI text. Focus on a compact object-level card titled “成交入账闭环” placed inside a renewal opportunity detail page, not a standalone landing page.

The card should show current opportunity amount, weighted amount, close status, and a concise billing adjustment preview. If not yet closed, show two inputs: “成交金额” and “入账说明”, plus a primary button “确认成交入账” with a receipt/check icon. Include a small warning note that the action will move the opportunity to won and create a billing adjustment source record. If already closed, show the linked adjustment number, signed amount, source type, created time, and a “查看调账记录” outline button. Include permission-denied and API error message areas as subtle bordered notices.

Visual style: minimal, technology-forward, product-grade, clean hierarchy, plenty of whitespace, no emoji, no cheap glow, no heavy gradients, no oversized round blobs. Use Tailwind/shadcn-like cards, status badges, compact form fields, and a confirmation modal overlay with Chinese labels “取消” and “确认入账”.
