# Product Prototype / Wireframe Prompt

Create a low-fidelity wireframe for a Chinese enterprise admin console route split.

Wireframe requirements:
- Left navigation supports multiple levels and collapse/expand behavior.
- `知识库中心` contains `活动总览`, `文档处理任务`, `召回记录`, `检索健康度`.
- `工具中心` contains `工具中心` list route and `执行记录`.
- `平台用量` contains `事件总览`, `平台事件`, `用量账本`, `用量趋势`, `事件告警`, `通知记录`, `治理任务`.
- The overview page includes only summary metrics, route cards, and lightweight relation/rollup cards.
- The focused list pages include title, refresh, filters, table, empty/error/loading placeholders, and a row detail action.

Do not include marketing copy or explanatory feature paragraphs. Use short Chinese operational labels only.
